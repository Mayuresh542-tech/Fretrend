# Fretrend 🚀

Fretrend is a **free, open-source** AI trend intelligence platform built for
content creators. Stop guessing what to create — let AI find what's about to go
viral in your niche.

## What Fretrend Does

- 🔥 Finds trending topics across your niche instantly
- 🤖 Generates complete AI Content Kits per trend
- ✍️ Writes full video scripts with timing
- 💡 Suggests viral titles, hooks & thumbnail ideas
- 📊 Scores each trend's virality from 0-100
- 💾 Save your favorite trend reports

## Built With

- Next.js 16
- Supabase
- Groq AI (free)
- Google Trends
- Framer Motion
- Tailwind CSS

## Getting Started

### Use the hosted app

1. Sign up free at fretrend.vercel.app
2. Get your free Groq API key at console.groq.com
3. Add key in Settings
4. Find your niche trends instantly!

### Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser. You'll need
a [Supabase](https://supabase.com) project and a free
[Groq API key](https://console.groq.com) configured via environment variables.
